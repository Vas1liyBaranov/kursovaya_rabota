# База данных торгового центра
Курсовая работа по дисциплине "Управление данными"

## о проекте
База данных для управления арендой торгового центра

## Структура БД
- Арендаторы
- Договоры
- Помещения
- Торговые точки
- Оплаты
- Бронирования
- Заявки на ремонт
- Отзывы

## Технологии
- Microsoft SQL Server
- Python + Streamlit
